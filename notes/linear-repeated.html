    Section 6 Solution Technique 4c: Linear Systems with Repeated Eigenvalues

    Sometimes you need to solve IVP of the form
    \begin{equation*} \vec{x}' = A\vec{x}, \vec{x}(t_0) = \vec{x}_0, \end{equation*}
    where \(A\) is a \(2\times 2\) matrix with only one eigenvalue and one linearly independent eigenvector. An example would be
    \begin{equation*} \begin{bmatrix} x'\\ y'\end{bmatrix} = \begin{bmatrix} 2 & -1 \\ 0 & 2 \end{bmatrix} \begin{bmatrix} 2 \\ y\end{bmatrix}. \end{equation*}
    In such a case, you can’t form two linearly independent solutions to build the generl solution. The most general solution we get is
    \begin{equation*} \vec{x}(t) = Ce^{2t}\begin{bmatrix} 1 \\ 0 \end{bmatrix}. \end{equation*}
    However, this system is partially decoupled and we can find the general solution
    \begin{equation*} \begin{bmatrix} x \\ y\end{bmatrix} = k_1e^{2t}\begin{bmatrix} 1 \\ 0 \end{bmatrix} + k_2\left(e^{2t}\begin{bmatrix} 0\\ -1\end{bmatrix} + te^{2t}\begin{bmatrix} 1\\ 0\end{bmatrix}\right). \end{equation*}
    It turns out all repeated eigenvalue systems have a general solution of this form, we just need to know how to find the vectors.
    🔗

    Theorem 6.2. General Solution with a Repeated Eigenvalue.

    Consider the first-order system of ODEs given by
    \begin{equation*} \vec{x}' = A\vec{x}, \end{equation*}
    where \(A\) is a \(2\times 2\) matrix with only one eigenvalue \(\lambda\) and one linearly independent eigenvector \(\vec{v}\text{.}\) Then the general solution to this system has the form
    \begin{equation*} \vec{x}(t) = k_1e^{\lambda t}\vec{v} + k_2e^{\lambda t}\left(\vec{v}_1 + t\vec{v}\right), \end{equation*}
    where \(\vec{v}_1\) is a nonzero vector that satisfies \((A-\lambda I)\vec{v_1} = \vec{v}\text{.}\)
    🔗
    🔗

    Proof.

    First, observe that \((A-\lambda I)\vec{v_1} = \vec{v}\) implies \(\lambda\vec{v_1} + \vec{v} = A\vec{v}_1\text{.}\) Hence we have
    \begin{align*} \frac{d}{dt}(e^{\lambda t}\left(\vec{v}_1 + t\vec{v}\right)) \amp = \lambda e^{\lambda t} \vec{v_1} + e^{\lambda t}\vec{v} + \lambda t e^{\lambda t}\vec{v}\\ \amp = e^{\lambda t}(\lambda \vec{v}_1 +\vec{v}) + te^{\lambda t} Av\\ \amp = e^{\lambda t}(A\vec{v_1}) + te^{\lambda t} Av \\ \amp = A(e^{\lambda t}\left(\vec{v}_1 + t\vec{v}\right)). \end{align*}
    Thus, the second part of the hypothesized general solution is a solution to the system. Moreover, because \(\vec{v}\) and \(\vec{v}_1\) are not zero, the two solutions are linearly independent at \(t=0\text{.}\) So, the generality of the solution follows.

    Solution Method 6.3. Eigenvector Method with Complex Eigenvalues.

    To solve
    \begin{equation*} \vec{x}' = A\vec{x}, \vec{x}(t_0) = \vec{x}_0, \end{equation*}
    when \(A\) has a repeated eigenvalue, proceed as follows:
    1. Find an eigenpair \((\lambda,\vec{v})\text{.}\) Then find a nonzero vector \(\vec{v}_1\) satisfying \((A-\lambda I)\vec{v_1} = \vec{v}\text{,}\) where \(I\) is the \(2\times 2\) identity matrix.
      🔗
      🔗
    2. Form the general solution
      \begin{equation*} \vec{x}(t) = k_1e^{\lambda t}\vec{v} + k_2e^{\lambda t}\left(\vec{v}_1 + t\vec{v}\right). \end{equation*}
      🔗
      🔗
    3. Solve for \(k_1\) and \(k_2\) using the initial conditions.
      🔗
      🔗
