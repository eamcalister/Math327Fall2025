    Section 4 Solution Technique 4a: Linear Systems with Distinct Real Eigenvalues

    The eigenvalue method for solving a system IVP of the form
    \begin{equation*} \vec{x}' = A\vec{x}, \vec{x}(t_0) = \vec{x}_0, \end{equation*}
    where \(A\) is a \(2\times 2\) matrix relies on two initial facts (they scale up for general \(n\times n\) matrices).
    🔗

    Theorem 4.1. The Superposition Theorem.

    If \(\vec{x}_1(t)\) and \(\vec{x}_2(t)\) are solutions to
    \begin{equation*} \vec{x}' = A\vec{x}, \end{equation*}
    then so is the linear combination
    \begin{equation*} c_1\vec{x}_1(t) + c_2\vec{x}_2(t) \end{equation*}
    for any constants \(c_1\) and \(c_2\text{.}\)
    🔗
    🔗

    Proof.

    (Sketch) This follows from the fact that the differential operator \(\frac{d}{dt}\) and matrix multiplication by \(A\) are both linear.
    🔗
    🔗

    Theorem 4.2. Non-Parallel Initial Values Create a Fundamental Solution Set.

    If \(\vec{x}_1(t)\) and \(\vec{x}_2(t)\) are solutions to
    \begin{equation*} \vec{x}' = A\vec{x}, \end{equation*}
    with \(\vec{x}_1(0)\) and \(\vec{x}_2(0)\) non-parallel (i.e. linearly independent), then the general solution to the system has the form
    \begin{equation*} c_1\vec{x}_1(t) + c_2\vec{x}_2(t) \end{equation*}
    That is, \(\{\vec{x}_1(t), \vec{x}_2(t)\}\) is a fundamental solution set.
    🔗
    🔗

    Proof.

    (Sketch) Observe that every vector is a linear combination of \(\vec{x}_1(t)\) and \(\vec{x}_2(t)\) because they are non-parallel.
    🔗
    🔗

    Subsection The Eigenvalue/Eigenvector Method

    🔗

    Solution Method 4.3. Eigenvector Method with Real Distinct Eigenvalues.

    To solve
    \begin{equation*} \vec{x}' = A\vec{x}, \vec{x}(t_0) = \vec{x}_0, \end{equation*}
    proceed as follows:
    1. Find the eigenpairs \((\lambda_1,\vec{v}_1)\) and \((\lambda_2,\vec{v}_2)\text{.}\) If \(\lambda_1\) and \(\lambda_2\) are real and distinct (not equal) proceed.
      🔗
      🔗
    2. Form the general solution
      \begin{equation*} \vec{x}(t) = c_1e^{\lambda_1 t}\vec{v_1} + c_2e^{\lambda_2 t}\vec{v_2} \end{equation*}
      🔗
      🔗
    3. Solve for \(c_1\) and \(c_2\) using the initial conditions.
      🔗
