    Section 5 Solution Technique 4b: Linear Systems with Complex Eigenvalues

    The eigenvalue method for solving a system IVP of the form
    \begin{equation*} \vec{x}' = A\vec{x}, \vec{x}(t_0) = \vec{x}_0, \end{equation*}
    where \(A\) is a \(2\times 2\) matrix with complex eigenvalues begins the same way as with real eigenvalues: first, find the eigenpairs using the Sage Cell below (same as previous, but pre-loaded with a complex eigenvalue matrix). From there, we need two key facts.

    Theorem 5.2. Eulerâ€™s Formula.

    The complex exponential is defined
    \begin{equation*} e^{(a+bi)t} = e^{at}(\cos(bt) + i\sin(bt)). \end{equation*}
    Using this definition, all properties of the exponential function that are relevant to differential equations are still true. Specifically,
    \begin{equation*} \frac{d}{dt}(e^{\lambda t}) = \lambda e^{\lambda t} \end{equation*}
    even if \(\lambda\) is complex.

    Theorem 5.3. The Real and Imaginary Parts of a Solution to the System are Solutions Themselves.

    Let
    \begin{equation*} \vec{x}_C(t) = \vec{u}(t) + i\vec{w}(t)\text{,} \end{equation*}
    where \(\vec{u}(t)\) and \(\vec{w}(t)\) are real-valued. That is,
    \begin{equation*} \vec{u}(t) = \mathrm{Re}(\vec{x}_C(t)) \end{equation*}
    and
    \begin{equation*} \vec{w}(t) = \mathrm{Im}(\vec{x}_C(t)) \text{.} \end{equation*}
    If \(\vec{x}_C(t)\) is a solution to
    \begin{equation*} \vec{x}' = A\vec{x}\text{,} \end{equation*}
    then so are \(\vec{u}(t)\) and \(\vec{w}(t)\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    (Sketch) Observe that
    \begin{equation*} \vec{x}'_C(t) = \vec{u}'(t) + i\vec{w}'(t) \end{equation*}
    and
    \begin{equation*} A\vec{x}_C(t) = A\vec{u}(t) + iA\vec{w}(t). \end{equation*}
    Since \(\vec{x}'_C(t) = A\vec{x}_C(t)\text{,}\) we may equate real and imaginary parts of right-hand sides of the above equations to obtain the result.
